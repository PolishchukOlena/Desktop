Види селекторів:

  1. Універсальний селектор

    Цей селектор призначений для вибору всіх елементів. В css використовується символ "*".

  2. Селектор за назвою тега

    Цей селектор використовується для надання стилю всім елементам відповідного типу (заголовки, абзаци і тд). Для створення цього селектору потрібно помістити ім'я елемента перед визначенням стилю.

  3. Селектор по id

    З його допомогою можна привласнювати стилі елементам, які мають вказане ім'я ідентифікатора. В css використовується символ "#".

  4. Селектор за класом

    З його допомогою можна привласнювати стилі елементам, які мають вказане ім'я класу. В css використовується символ ".".

  5. Селектор з атрибуту

    Цей селектор вибирає всі елементи, що мають цей атрибут або атрибут з певним значенням.

  6. Селектор за кількома класами

    Ці селектори використовуються для того, щоб обрати елементи, які одночасно задовольняють відразу декількам умов. Селектори, застосовувані до одного елементу, пишуться в css без пробілів.

  7. Дочірні селектори

    Дочірній селектор присвоює стиль тільки одному елементу, безпосередньо дочірньому. В css позначається символом ">".

  8. Вкладені селектори

    Ці селектори використовують для того, щоб застосувати стилі до елементу, тільки якщо він вкладений в інший потрібний елемент. Селектор складається з декількох частин, розділених пробілом

  9. Суміжні селектори

    За допомогою цих селекторів можна привласнити стиль елементу, який передує зазначеному. Суміжний і зазначений елементи повинні мати один і той же батьківський елемент. В css позначається символом "+".

  10. Псевдокласи

    Псевдокласи - це доповнення до звичайних селекторів, які роблять їх ще точнішими. Псевдокласи визначають динамічний стан елементів, що змінюється за допомогою дій користувача, а також положення в дереві документа.
